# 🧩 Odin Pokémon Tracker

A full-stack Pokémon display app using **React**, **Express**, **REST APIs**, and **PostgreSQL**. It fetches data from the [PokéAPI](https://pokeapi.co/) and stores it locally to allow efficient display and manipulation.

---

## 📦 Tech Stack

| Layer       | Tech                      |
|-------------|---------------------------|
| Frontend    | React + Vite              |
| Backend     | Node.js + Express         |
| API Source  | [PokéAPI](https://pokeapi.co/) |
| Database    | PostgreSQL                |

---

## 🚀 Features

- Fetches and stores data for the first 50 Pokémon
- Displays Pokémon names and IDs
- Full RESTful API for fetching local Pokémon data
- Modern React frontend styled for clarity
- Primitive(not safe, it's with plan text and no encryption yet!) profile storage
